# N-Back Chess

Play chess against a bot while training your working memory. Every move
gets a letter, spoken aloud. Occasionally you'll be asked to recall the
letter from N moves ago **and** reconstruct what the board looked like
back then.

**[Play it here](https://vegetashanks911.github.io/n-back-chess/)**

## Running locally

No build step needed — just open `index.html` in a browser, or serve the
folder with any static file server.
